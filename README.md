# Tank Battle

It is a online multiplayer game for two players

Run the following commands to get started:
```
git clone git@gitlab.engr.illinois.edu:mgautam2/sp21-cs242-project.git
cd sp21-cs242-project.git
```

# Project Structure

FrontEnd

```
	npm i
	npm start
```

BackEnd
```
	npm i
	npm start
```
